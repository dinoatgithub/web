color: hexadecimal_number | rgb | rgba | name
specifies text color
hexadecimal_number can be generated by putting color: #ffffff;
now using color picker can be adjusted
the color picker helps to get the rgb value where each of rgb can be anything from 0-255
rgba has 4th component as opacity ranging from 0-1
color names are not recommended as the shade cannot be implied
always use hexadecimal_number
