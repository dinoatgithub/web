Targets a group of elements based on class name. E.g. all cities or all weathers.
Syntax.

    .className{
        property:value;
    }

Class means similar elements.
Class can be duplicated in the web page

.container{}
.city{}
.weathers{}

ID selector overwrites the Class selector.

Apply class name to the elements.
